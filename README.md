## 内容

- `SKILL.md` — OpenClaw Skill（AI转型成功率测评构建器）
- `push-to-github.sh` — 推送脚本（需要 GITHUB_TOKEN）
- `package.sh` — 打包脚本

## 快速部署

```bash
# 克隆后在任意 OpenClaw 实例安装 skill
# 将 SKILL.md 放到 ~/.openclaw/skills/ai-assessment-builder/

# 或部署 React 版本
cd ai-assessment-app
npm install
npm run build
# dist/ 目录部署到你的服务器
```

## 功能

- 15道情境题（AI认知、战略适配、资源执行、流程适配、心态韧性）
- McKinsey级报告（雷达图、30-60-90天路线图、Top5行动、工具包、风险预警）
- React + Vite + Tailwind + Recharts
