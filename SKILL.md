# AI Assessment Builder

Builds a professional AI transformation success rate assessment for individual entrepreneurs.

## Tech Stack

- React 18 + TypeScript + Vite
- Tailwind CSS (utility classes, gradient headers, rounded-2xl cards)
- Recharts (RadarChart for 5-dimension visualization)
- Deploy via `deploy` tool

## Project Location

Source: `/workspace/ai-assessment/`
Dist (built): `/workspace/ai-assessment/dist/`
Live: `https://3kn60nw8nii1.space.minimaxi.com`

## Build Commands

```bash
cd /workspace/ai-assessment
node_modules/.bin/vite build
```

Note: `pnpm install` fails (symlink permission). Vite and TypeScript already exist in `node_modules/.bin/`. Skip install, run build directly.

## Key Files

- `src/App.tsx` — Full React app (intro → quiz → results flow)
- `src/index.css` — Tailwind directives only
- `dist/index.html` — Auto-generated by Vite build

## Core Architecture

### State Machine

```typescript
type Step = 'intro' | 'loading' | 'quiz' | 'results';
// step='loading' shows star animation + "正在生成报告" for 1.5s
// Quiz auto-advances on option select (250ms delay)
// Quiz auto-submits on last question option select
```

### 5 Dimensions (Questions 1-3 each)

| Dim | Name | Icon | Color |
|-----|------|------|-------|
| 1 | AI认知 | 🤖 | #1565c0 |
| 2 | 战略适配 | 🔥 | #c62828 |
| 3 | 资源执行 | 🧱 | #6a1b9a |
| 4 | 流程适配 | ⚙️ | #2e7d32 |
| 5 | 心态韧性 | 🧘 | #e65100 |

### Scoring

- Each answer: 1–5 points
- Per dimension: average → convert to 10-point scale
- Overall: weighted average across all 5 dimensions

### Report Sections (Results page)

1. Score ring (SVG, animated)
2. Grade tag (S/A/B/C/D based on score)
3. Radar chart (Recharts RadarChart)
4. Dimension bars (5 color-coded cards)
5. Core strengths (dims ≥ 7)
6. Key weaknesses (dims < 6)
7. 30-60-90 day roadmap
8. McKinsey-level strategic insights
9. Top 5 priority actions (dimension-tagged)
10. Tool recommendations (industry + skill-level filtered)
11. Common failure mode warnings
12. Expert tips (McKinsey/Tencent/36Kr quotes)

### Action Prioritization Logic

```typescript
// For each dimension score:
// < 5: add action with priority 1-2
// 5-7: add action with priority 2-3
// >= 7: add advanced action with priority 3
// Sort by priority, fill to 5 items with fallback actions
```

## Customization

### Change Colors
Edit `DIM_COLORS` array and gradient classes (`from-rose-400 to-pink-500`).

### Change Questions
Edit `QS` array in `App.tsx` — each question has: `cat`, `dim` (1-5), `icon`, `name`, `q`, `scenario`, `opts: [text, score][]`.

### Add Industry Tools
Add entry in `getTools()` → `indTools` object.

## Assets
